## Usage

1. Clone project code.

```bash
git clone git@github.com:mirrorworld-universe/mirrorworld-sdk-rust.git
cd mirrorworld-sdk-rust
```

2. run.

```bash
cargo run 
```

1. test.

```bash
cargo test
```
